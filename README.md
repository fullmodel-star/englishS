# 🧩 國中英語句型（多鄰國式拼句）

一個**離線可用、免安裝、零追蹤**的英語**拼句**練習 PWA。仿多鄰國（Duolingo）玩法：把打散的單字方塊依中文提示拖成正確英文句子，從國小到國中循序闖關。

> 純單一 HTML 檔 + 題庫 `bank.json`、資料全存本機、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　☁️ 天空藍主題（theme `#1f8fd6`）　｜　**國小 → 國中 7 關 / 63 單元 / 630 題**　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/englishS/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)

---

## ✨ 功能
- **拼句闖關**：依中文提示，把單字方塊拖成正確句子；含干擾字（`bank.json` 的 `d` 欄）
- **關卡分級**：國小基礎 → 國中，共 **7 關、63 單元、630 題**
- **提示鈕**：卡關可看提示
- **長按發音**：長按單字／句子朗讀（TTS）
- **錯位紅框**：放錯位置即時紅框標示
- **愛心生命值 + 愛心彈窗**：答錯扣愛心、過關愛心動畫
- **emoji 情境圖示**：每單元有主題 emoji

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（拼句遊戲） |
| `bank.json` | 題庫：關 → 單元 → 題（`zh` 中文、`t` 正解字序、`d` 干擾字） |
| `_shell.html` | 版型／外殼（開發用） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | 圖示 |

## 🔒 隱私
完全離線、不連外部伺服器；進度只存裝置 `localStorage`；無帳號、無追蹤、無廣告。適合未成年使用。
